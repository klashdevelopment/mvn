# mvn repo
## The Public KlashDevelopment Maven Repository!

### Using Maven
Implement:
```xml
<repository>
   <id>klashdevelopment</id>
   <url>https://raw.githubusercontent.com/klashdevelopment/mvn/main/repository/</url>
</repository>
```
